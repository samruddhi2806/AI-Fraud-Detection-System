# AI Fraud Detection System using Machine Learning

## Project Overview
This project is a Machine Learning-based Fraud Detection System that detects
whether a credit card transaction is fraudulent or legitimate.

## Dataset
- Total Records: 284,807
- Total Features: 30
- Target Variable: Class (0 = Legitimate, 1 = Fraud)
- Source: Kaggle Credit Card Fraud Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Model Used
Random Forest Classifier

## How to Run the Project

1. Clone the repository
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run Streamlit app:
   python -m streamlit run app.py

## Output
The system predicts whether a transaction is fraudulent or legitimate using
an interactive web interface.
